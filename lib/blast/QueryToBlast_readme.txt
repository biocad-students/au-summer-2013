Для того, чтобы воспользоваться запросом к бласту необходимо указать 5 параметров
1 файл с последовательностью, по которой хотим искать
2 файл куда выведется результат
3 taxid или их список через пробел по которым хотим фильтровать или all
4 1 для тяжелых цепей, 2 для лёгких, 0 для неизвестных или их комбинацию или all
5 1 IgG    2 IgM   3 IgA     4 IgD     5 IgE, 0 для неизвестных или их комбинацию или all


С параметрами фильтра, которые указали на входе, программа обратиться к файлу, где хранятся весь список GI, что у нас есть, с данными о таксах типах цепей и антител, через модель UseFilter. На выходе из него она получает список GI, которые надо подставить в запрос к бласту. Выполняет этот запрос, результат загружает в xml файл, который затем разбирает, и нужную инфу записывает в фаста файл.
